swiper (轮播图)

属性 类型 默认值 说明
indicator-dots bololean false 是否显示面板指示点
indicator-color color rgba(0,0,0,3) 指示点颜色
indicator-active-color color #000000 当前选中的指示点颜色
autoplay boolean false 是否自动切换
interval number 5000 是否自动时间间隔
circular boolean false 是否采用街接滑动 
<!--pages/login/login.wxml-->
<swiper class="swiper-container" indicator-dots indicator-color="gray" autoplay interval="3000" circular>
<!-- autoplay是否自动切换 interval 自动时间间隔ms  circular是否滑动衔接  -->
